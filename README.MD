# Fixup185545

Fixes [MC-185545](https://bugs.mojang.com/browse/MC-185545). (AKA Netherite random knockback bug)

This plugin is designed for **Spigot 1.16.5 -> 1.18.1**.

This plugin has **no effect for Spigot 1.18.2 and later**.
